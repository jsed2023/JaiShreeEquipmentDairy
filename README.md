
# Next.js & NextUI Template

This is a template for creating applications using Next.js 14 (app directory) and NextUI (v2).

## Technologies Used
- Next.js 14
- NextUI v2
- Tailwind CSS
- Tailwind Variants
- TypeScript
- Framer Motion
- next-themes

## How to Use

### Install dependencies
```
npm install
```

### Run development server
```
npm run dev
```

### Build
```
npm run build
```

### Start (Production)
```
npm start
```

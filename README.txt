# JaiShreeEquipmentDairy-main

मैंने आपके लिए यह `package.json` फ़ाइल बनाई है ताकि आप अपना project चलाना शुरू कर सकें.

## आगे क्या करना है (Windows)
1. इस फोल्डर को अपने कंप्यूटर पर unzip या सीधे इस `package.json` को अपने project folder में रखें।
2. Project folder खोलें और terminal खोलें (Windows Terminal या PowerShell).
3. Folder में जाएँ:
   ```
   G:
   cd JaiShreeEquipmentDairy-main
   ```
4. Commands चलाएँ:
   ```
   npm install
   npm run build
   npm start
   ```
5. फिर ब्राउज़र खोलें: http://localhost:3000

अगर कोई error आए तो error message या screenshot भेज दें — मैं तुरंत मदद कर दूँगा।
